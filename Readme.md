# HourGlass Plugin

A hackable hourglass for your next web project.



## Pre-Requirements

- NodeJS



## Quick Start

Here are the steps to get you started asap.

1. Install the necessary npm packages

```
npm install
```

2. Transpile the latest scss to css

```
npm run build-scss
```

3. Run the demo

```
npm start
```
